# My Content xN

A TypeScript-based content management project with integrated assets and styling.

## Overview

This repository contains a modern content management solution built with TypeScript. It includes custom fonts, images, and a well-structured architecture for content handling.

## Features

- 🎯 **TypeScript Support** - Fully typed codebase for better development experience
- 🎨 **Custom Fonts** - Space Mono font included for consistent typography
- 🖼️ **Asset Management** - Organized image and font assets
- 📦 **Modular Architecture** - Clean separation of concerns

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/Mwesige256/my-content-xN.git

# Navigate to the project directory
cd my-content-xN

# Install dependencies
npm install
```

### Usage

```bash
# Development server
npm run dev

# Build
npm run build

# Run tests
npm test
```
